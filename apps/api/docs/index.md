# Fridge App Backend API

Backend API documentation for the Fridge App.

## Development

Run the backend locally:

```sh
uv run poe api --dev
```

Run tests:

```sh
uv run poe test
```

Run database migrations:

```sh
uv run poe migrate
```

## API Docs

When the app is running, open `/docs` for the FastAPI Swagger UI.
